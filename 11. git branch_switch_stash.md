# 1. `git branch`

Sirve para ver, crear o eliminar ramas.

Muestra todas las ramas locales (la actual tiene `*`).

## Crear una rama

```

git branch nombre-rama

```
Solo la crea, **NO** nos mueve a ella.

## Eliminar una rama

```

git branch -d nombre-rama

```

# 2. `git switch`

Sirve para cambiar de rama (movernos entre ramas).

- Es la forma moderna (más clara que `git checkout`).

## 2.1. Cambiar de rama

```

git switch nombre-rama

```

## 2.2. Crear y cambiar en un solo paso

```

git switch -c nombre-rama

```

# 3. `git stash`

Sirve para guardar cambios temporales sin hacer commit y para movernos entre ramas.

- Es como “poner en pausa” lo que estamos haciendo.

## 3.1. Guardar cambios

```

git stash

```

## 3.2. Ver stashes guardados

```

git stash list

```
## 3.3. Recuperar cambios

```

git stash pop

```

Aplica los cambios y los elimina del stash.


## 3.4. Aplicar sin borrar

```

git stash apply

```
