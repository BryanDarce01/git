# `git commit --amend`

Se usa para modificar el último commit.

### Ejemplo

```

git add archivo.py
git commit --amend

```

Git hace:

- toma el último commit

- agrega los nuevos cambios

- reescribe ese commit

También puedes cambiar solo el mensaje:

```

git commit --amend -m "Nuevo mensaje"

```

Muy usado cuando:

- se olvidó agregar un archivo

- el mensaje del commit está mal

- todavía no has hecho push

⚠️ No se recomienda si el commit ya está en el repositorio remoto.
